Belajar HTML & CSS

Diposting pada 10 September 2025 oleh Admin

HTML (HyperText Markup Language) adalah bahasa dasar untuk membuat struktur halaman web. Dengan HTML, kita bisa menambahkan teks, gambar, tabel, form, hingga video ke dalam halaman.

CSS (Cascading Style Sheets) digunakan untuk mempercantik tampilan web. Misalnya mengatur warna, ukuran font, tata letak, dan efek animasi.

"Menguasai HTML & CSS adalah pondasi sebelum melangkah ke dunia JavaScript dan framework modern."

Jadi, jangan terburu-buru belajar framework kalau dasar ini belum kuat. Mulailah dengan proyek kecil, misalnya membuat personal portfolio atau blog sederhana.
